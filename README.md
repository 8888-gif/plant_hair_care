# 青禾植染植物养发项目

这是一个基于 Next.js 的植物养发门店展示页面，面向植物染发、遮白染、头皮养护和到店预约场景。页面包含服务项目、发色色卡、养护流程、顾客评论滚动模块、门店地址和预约表单。

## 功能内容

- 首页视觉展示：植物染发养护空间介绍和主推服务。
- 服务项目：草本遮白染、植物栗棕染、头皮净养护理。
- 色卡展示：黑茶棕、栗子棕、深木棕、暖茶棕。
- 流程说明：头皮评估、配方调色、分区上色、染后养护。
- 评论滚动：展示近期顾客体验反馈。
- 预约区域：门店地址、营业时间、联系电话和预约信息表单。

## 技术栈

- Next.js 15
- React 19
- CSS 响应式布局
- next/image 图片优化

## 本地运行

安装依赖：

```bash
npm install
```

启动开发服务：

```bash
npm run dev -- -p 3000
```

浏览器访问：

```text
http://127.0.0.1:3000/
```

## 常用命令

代码检查：

```bash
npm run lint
```

生产构建：

```bash
npm run build
```

启动生产服务：

```bash
npm run start
```

## 目录结构

```text
app/
  globals.css      全局样式
  layout.js        页面 metadata 和根布局
  page.js          首页内容和模块数据
public/assets/     页面使用的公开图片资源
assets/            原始图片资源备份
scripts/           Next.js 启动辅助脚本
```

## 注意事项

- 不要提交 `node_modules`、`.next`、日志文件等本地生成内容。
- 开发服务运行期间不要同时执行生产构建，避免 `.next` 缓存被不同模式写入。
- 如果本地页面出现缓存相关 500 错误，可停止开发服务后重新生成 `.next`。
- 后续改进可以添加后端
